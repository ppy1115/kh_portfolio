🔥 해결한 문제

경영진이 필요한 정보를 찾지 못하는 문제

  • 급여대장 엑셀에서 수작업으로 필터링하며 데이터 찾기
  • 전월 대비 변동 사항을 일일이 비교
  • 과표/공제 차이 나는 직원을 수동으로 찾아야 함

  • 기본급/역할급 변동자를 찾기 위해 VLOOKUP 반복

결과: 의사결정이 느려지고, 임원이 직접 데이터 분석에 시간 소비

인터랙티브 대시보드 시스템

임원레포트는 클릭 기반 드릴다운, 총 8개 모듈로 구성됩니다:

Excel VBA 이벤트 처리 (DoubleClick) Dictionary 고속 검색 동적 요약 테이블 전월 대비 분석

🎯 Stage 1: 데이터 준비 (3개 모듈)

급여대장 데이터를 검증하고 분석 준비를 완료합니다.

모듈 1 ▼

초기화 및 검증

기준/예외자/KEY 데이터 검증 후 시트 초기화

3단계 검증 시스템

Step 1: TO-BE 시트 검증
- "기준" 표에 회사명, 기준일 입력 확인
- 기준일에서 연월(yyyy-mm) 추출

Step 2: 예외자 시트 검증
- "종료일" 헤더 확인
- 종료일이 기준 연월 이전인 행 경고
- 사용자 선택: 무시 or 중단

Step 3: KEY 시트 검증
- "명부대사1120212" 표 존재 확인
- "기준월" 컬럼에서 일치하는 월 찾기
- 일치 시 "메시지" 열 내용 표시

초기화 작업

  • 시트 정리: 필요한 13개 시트만 남기고 삭제
  • 이름 변경: "당월" → "전월"로 자동 변경
  • 세금검증: 사번/당월세율 → "임시" 시트 백업
  • 데이터 삭제: 세금검증 시트 2행 이하 전체 삭제
모듈 2 ▼

파일 불러오기

Input 테이블 기반 다중 파일 검증 및 시트 가져오기

사전 검증 시스템

1. 사용자가 여러 파일 선택 (MultiSelect)
2. Input 테이블에서 파일명 확인
3. 하나라도 없으면 전체 작업 중단
4. 모든 파일 검증 완료 후 가져오기 시작

→ 잘못된 파일로 작업하는 실수 원천 차단!

자동 처리

  • 확장자 제거하여 Input 테이블과 비교
  • 파일명 → 시트명 매핑 (Trim 처리)
  • 동일 시트명 있으면 자동 삭제 후 재생성
모듈 3 ▼

인원검증

KEY 시트 설정 기반 검증 시트 자동 생성

동적 시트 생성

KEY 시트 "인원검증" 표 구조:
헤더명 | 참고시트 | 참고열 | 수식

예시:
사번 | 급여대장_당월 | A | (비어있음)
성명 | 급여대장_당월 | B | (비어있음)
비교 | | | =A2=VLOOKUP(A2,급여대장_전월!A:B,1,0)

→ 수식 있으면 수식 입력, 없으면 참고열 값 복사

자동 정렬 (FALSE 우선)

"비교" 헤더 열 기준으로:
1순위: FALSE (차이 나는 사람)
2순위: 오류값 (#N/A, #VALUE! 등)
3순위: TRUE (일치하는 사람)

→ 문제 있는 데이터가 맨 위로!

📊 Stage 2: 항목별 검증 (5개 모듈)

과표/공제/보험/세금을 전월과 비교하여 차이를 자동 검증합니다.

모듈 4 ▼

지급 가공

급여대장에서 지급 관련 항목 추출 및 가공

급여대장의 복잡한 데이터에서 지급 항목만 추출하여 분석 가능한 형태로 변환합니다.

모듈 5 ▼

과표,공제 시트 생성

과세표준/공제 항목을 전월 대비 자동 비교

자동 차이 계산

각 항목별 (과세표준, 건강보험, 국민연금 등):
당월 합계 - 전월 합계 = 차이

차이 ≠ 0인 항목만 표시
→ 변동 없는 항목은 숨김 처리

인터랙티브 기능

더블클릭 이벤트: "차이" 열을 더블클릭하면 해당 항목에서 차이 나는 사람들의 상세 내역이 3개 열 옆에 자동 표시됩니다.

표시 항목:
- 사번
- 성명
- 직위
- 당월금액
- 전월금액

한 번 더 더블클릭하면 상세 내역 삭제!
모듈 6-8 ▼

보험/세금/세후 검증

4대보험, 세금, 세후 금액 자동 검증

  • 보험작업: 건강보험, 국민연금, 고용보험, 산재보험 검증
  • 세금검증: 소득세, 지방소득세 자동 계산 및 검증
  • 세후검증: 실수령액 = 지급총액 - 공제총액 확인

📈 Stage 3: 요약 및 대시보드

경영진이 바로 볼 수 있는 인터랙티브 대시보드를 생성합니다.

대시보드 ▼

기본요약_당월 (인터랙티브)

클릭 한 번으로 상세 내역 드릴다운

5개 인터랙티브 테이블

1. 인원 차이 (D열 클릭)

당월 vs 전월 인원 차이 클릭
→ 차이 나는 사람들 상세 내역 표시
→ 한 번 더 클릭하면 삭제

2. 실수령액 구간별 인원 (C열 클릭)

구간별 인원 수 클릭:
- 급여 이상 (300만원 이상)
- 급여 300 (200~300만원)
- 급여 200 (100~200만원)
- 급여 100 (100만원 미만)

→ 해당 구간에 속한 직원 목록 표시

3. 전월 대비 상이자 (C열 클릭)

직위별 상이자 수 클릭
→ 해당 직위의 상이자 명단 표시
→ 기본급/역할급 비교 포함

4. 기본급/역할급 변동 (F/G열 클릭)

F열 (기본급 변동) 클릭
→ 기본급이 변동된 직원 목록

G열 (역할급 변동) 클릭
→ 역할급이 변동된 직원 목록

5. 사대보험 변동/차이 (J/K열 클릭)

J열 (변동자) 클릭
→ 보험료가 변동된 직원

K열 (차이자) 클릭
→ 보험료 차이 발생 직원

핵심 성공 요인

1. 인터랙티브 UX

  • 더블클릭 이벤트: 숫자 클릭 → 상세 내역 즉시
  • 토글 방식: 같은 곳 다시 클릭 → 삭제
  • 시각적 구분: 진한회색(항목명), 옅은회색(헤더)
  • 자동 정렬: FALSE 우선 → 문제 있는 데이터 먼저

2. 지능형 검증

  • 사전 검증: 파일 불러오기 전 Input 테이블 대조
  • 3단계 검증: 기준 → 예외자 → KEY 순차 확인
  • 자동 경고: 종료일이 기준 연월 이전인 경우 알림
  • Dictionary 검색: 사번 기반 빠른 매칭

3. 유연한 설정

  • KEY 시트 기반: 코드 수정 없이 테이블만 수정
  • 동적 헤더 찾기: "비교" 포함 열 자동 인식
  • Input 테이블: 파일명-시트명 매핑 중앙 관리

🔧 기술 상세

사용 기술

Excel VBA 이벤트 처리 (DoubleClick) Dictionary (해시맵) 동적 범위 처리 ListObjects (표) Find / Match 정규식 (월 추출)

핵심 기술 포인트

1. 이벤트 기반 인터랙션
Workbook_SheetBeforeDoubleClick 이벤트
→ 클릭한 셀의 위치/값에 따라 동작 분기

2. Dictionary 고속 검색
사번 → 금액/성명/직위 매칭
검색 속도: O(1)

3. 동적 헤더 찾기
"비교" 단어 포함 열 자동 인식
→ 표 구조 변경에도 자동 대응

4. 정규식 활용
"1월", "11월" → 숫자만 추출 (ExtractMonthNumber)
→ 기준월과 KEY 데이터 월 비교

확장 가능성

  • 새 검증 항목: KEY 시트 "인원검증" 표에 행 추가
  • 새 파일 추가: Input 테이블에 파일명-시트명 추가
  • 새 이벤트: Workbook_SheetBeforeDoubleClick에 분기 추가

💡 배운 점

사용자 경험 (UX)의 힘

  • ✅ 클릭 한 번의 마법: 복잡한 필터링 → 더블클릭
  • ✅ 즉각적 피드백: 클릭 즉시 결과 표시
  • ✅ 직관적 인터페이스: 숫자 보고 클릭하면 상세 나옴
  • ✅ 토글 방식: 같은 곳 다시 클릭하면 정리됨

이벤트 기반 프로그래밍

  • ✅ 사용자 행동 감지: DoubleClick, Change 등
  • ✅ 조건부 분기: 클릭한 위치에 따라 다른 동작
  • ✅ Cancel 플래그: 기본 동작 차단 후 커스텀 동작
